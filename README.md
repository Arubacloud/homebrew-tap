# Homebrew Tap — Aruba Cloud CLI

This is the official [Homebrew](https://brew.sh) tap for the [Aruba Cloud CLI](https://github.com/Arubacloud/acloud-cli).

## Install

```bash
brew tap Arubacloud/tap
brew install acloud
```

## Upgrade

```bash
brew update
brew upgrade acloud
```

The formula in `Formula/acloud.rb` is updated automatically on every release of [acloud-cli](https://github.com/Arubacloud/acloud-cli).
